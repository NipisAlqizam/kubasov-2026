# Работа с тензорами 4 ранга (механические)
Запускаются файлы `s_tensor.m` (в нём керамика PZT) и `quartz.m` (в нём кристаллы кварца).

Остальные файлы — функции, переделанные из файлов для пьезомодулей.

- `s_matrix_into_tensor.m`  — перевод матриц $S^E$ в соответствующиц тензор
- `stensor2matrix.m` — перевод тензора $S^E$ обратно в матрицу
- `s_tensor_rotate.m` — поворот тензора в лоб
- `smatrix_rotate.m` — поворот через матрицу
