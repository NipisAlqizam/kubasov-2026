# Пьезомодули
Это файлы, на основе того, что было у Зои. Запускаются `PZT.m` (PZT) и `LiNb.m`) (ниобат лития). Остальные — файлы с функциями.

В ниобате лития ему не понравилась первая картинка, на следующей паре (7 апреля) он обещал посмотреть.

- `d_matrix_into_tensor.m` — перевод тензора пьезомодулей в матрицу
- `dtensor2matrix.m` — перевод матрицы обратно в тензор
- `d_tensor_rotate.m` — поворот тензора в лоб
- `d_tensor_d33_rotate.m` — поворот только элементов d33 в лоб
- `d_tensor_rotate_fast.m` — оптимизированный варик через матричные перемножения
- `dmatrix_rotate.m` — итоговый вариант без перехода к тензору

## Про ниобат лития
Из-за того, что поворачиваем вокруг X1 и X2, особая ось - X1 и брать надо не d33, а d11.

Если строим поверхность, то строим ее вокруг индекса, соответствующего направлению первого поворота.
